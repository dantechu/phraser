**Last Updated:** October 16, 2025

## Habit Builder Categories Overview

The Habit Builder supports 10 main categories:

1. **Health & Fitness** - Physical wellness and exercise
2. **Mind & Emotions** - Mental health and emotional wellbeing
3. **Learning & Growth** - Personal development and knowledge
4. **Productivity & Work** - Focus and professional development
5. **Finance & Money** - Financial health and money management
6. **Lifestyle & Routine** - Daily routines and lifestyle habits
7. **Relationships & Social** - Social connections and relationships
8. **Creativity & Hobbies** - Creative pursuits and hobbies
9. **Contribution & Impact** - Community and helping others
10. **Spirituality & Mindfulness** - Spiritual practices and mindfulness

## Recommendations for Implementation

### 1. Dynamic Quote Selection
When a user creates or tracks a habit, the app should:
- Identify the habit's category (e.g., Health & Fitness)
- Pull random quotes from mapped phraser categories (e.g., Running, Gym, Body Positivity)
- Display motivational quotes relevant to the specific habit

### 2. Tag-Based Filtering
Use the `tags` field in the Habit model to:
- Store comma-separated tags (e.g., "fitness,energy,strength")
- Filter phrasers by matching category names or tags
- Show contextually relevant quotes during habit tracking

### 3. Mood-Based Integration
Combine habit tracking with mood selection:
- If user selects "Motivated" mood → Show quotes from Success, Motivation categories
- If user selects "Peaceful" mood → Show quotes from Patience, Calm categories
- If user selects "Grateful" mood → Show quotes from Kindness, Self Love categories

### 4. Streak Milestones
Show special quotes when users hit milestones:
- 7-day streak: Success quotes (28)
- 30-day streak: Personal Growth quotes (13)
- 100-day streak: Wisdom quotes (38)

### 5. Category Customization
Allow users to customize which phraser categories feed into their habit:
- Default to mapped categories
- Let users add/remove categories based on preference
- Save customization in the Habit's `tags` field

## API Integration Example

```dart
// Get quotes for a specific habit
Future<List<Phraser>> getHabitMotivationalQuotes(Habit habit) async {
  final tags = habit.tags?.split(',') ?? [];
  final category = habit.categoryEnum;

  // Map habit category to phraser categories
  List<int> categoryIds = [];

  switch (category) {
    case HabitCategory.healthFitness:
      categoryIds = [44, 11, 42, 36]; // Running, Gym, Body Positivity, Healthier Life
      break;
    case HabitCategory.mindEmotions:
      categoryIds = [50, 45, 43, 40, 37, 12, 41, 35, 51];
      break;
    case HabitCategory.learningGrowth:
      categoryIds = [47, 29, 28, 18, 13, 38];
      break;
    // ... other categories
  }

  // Fetch phrasers from database by category IDs
  final phrasers = await database.phraserDAO
      .getPhrasersByCategories(categoryIds);

  return phrasers..shuffle(); // Randomize
}
```

## Notes

1. **Multi-Category Support:** Some phraser categories (like Success, Wisdom, Motivation) can be used across multiple habit categories
2. **Premium Content:** 11 out of 38 categories are Paid, providing additional motivation for premium users
3. **Flexibility:** Users can create custom habits and manually select which phraser categories to draw from
4. **Dynamic Updates:** As new phraser categories are added via API, they can be mapped to existing habit categories
5. **Regional Support:** Category fetching supports `&region=` parameter for localized content
